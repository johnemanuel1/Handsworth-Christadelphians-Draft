# Handsworth Christadelphians website draft

This repository contains a public draft for review before the new website goes live.

## Review notes

- The Members area, private talks, address roll and private calendar are deliberately excluded.
- Search engines are discouraged from indexing this review copy.
- The final Namecheap domain has not been connected.
- Canonical website metadata still refers to the intended final public domain.

## Suggested review areas

- Accuracy of beliefs, meeting times, address and contact information
- Clarity and welcome for first-time visitors
- English and Farsi content
- Mobile readability
- Broken links, missing images and wording corrections
